# 支持游戏

## 恐龙快打

出招表：
- ⬇⬆A: q
- 连发：j (按住不放)

## 西游释厄传

出招表：
- 升龙斩：q
- 元神出窍：e/r
- 变小：o
- 遁入幻境：v
- 乾坤波动功：f
- 看宝A兵：y
- 升龙斩接翻腾舞：q+n/m

> 升龙斩接翻腾舞：需要把握n/m释放的时机，如果嫌麻烦直接(⬇➡A)或(⬇⬅A)

> 将来会加入**乾坤十连斩**
